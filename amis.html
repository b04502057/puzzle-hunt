概況

阿美(Amis)是阿美族人的自稱，涵義為"北"。依據語言、習俗和地域差異大致可分為北、中、南部阿美和海岸阿美四種。
阿美語屬於南島語系(Austronesian)，主要特色如下 :
(一) 18個子音，5個母音，沒有雙子音、濁塞音、濁塞擦音、濁擦音、送氣音和雙母音。重音在詞的最後音節上。
(二)詞的型態變化主要藉由詞根加綴達成。有前綴、中綴和後綴，其中前綴最多，後綴其次，中綴最少。詞幹除了部分原始詞根外，
大部分是帶有詞綴的衍生詞幹。阿美語還有豐富的重疊詞，重疊詞因為詞根音節結構不同會代表不同的涵義。
(三)阿美語是VSO型語言，即verb, subject, object，謂語(動詞)主詞受詞。修飾時修飾語放在被修飾語之前；被擁有的物品放在
擁有者之前。各種詞在句中的關係是藉由格助詞或是指示代名詞來改變意義。動詞謂語句有主動態、被動態和處置態。

子音特點

1. 只有不送氣音，沒有送氣音。塞音、塞擦音和擦音只有清音沒有濁音。沒有複子音。
2. 18個子音都可以出現在詞首、詞中和詞尾。半母音y和w不常出現於詞首。舌面後擦音x較少出現。
3. 塞音、塞擦音和擦音出現於詞尾時以清音收尾，例如
layap [layap(h)] 伸手
ritrit [ritrit(h)] 割草
tapelik [tapelik(h)] 浪濤
tama' [tama'] 獵獲物
teroc [teroc(h)] 跳躍
dihif [dihif(h)] 山洞
toros [toros(h)] 膝蓋
saperox [saperox(h)] 鑽子
kapah [kapah(h)] 年輕人
4. 鼻音出現詞尾時讀音略帶延長，例如
nanom [nanom:] 水
futing [futing:] 魚

句子結構

句型

a. 謂語 + 主格 = {動詞、形容詞等} + {ko(主格助詞) + 名詞}

b. 謂語 + 主語 + 賓語 = {動詞、形容詞等} + {ko(主格助詞) + {to(受詞助詞) + 名詞}

c. 主語 + 謂語 = {o(判斷助詞) + 名詞} + {動詞、形容詞等}

d. 主語 + 謂語 + 受格 = {o(判斷助詞) + 名詞} + {動詞、形容詞等} + {to(受詞助詞) + 名詞}
